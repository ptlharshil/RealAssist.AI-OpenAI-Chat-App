# RealAssist.AI-OpenAI-Chat-App

Real Assist is an AI enabled conversational application to answer any kind of questions. It is built using OpenAI API, React (Client), Node (Server) and MongoDB. Multiple chats can be created or deleted. Chat titles can be edited and each chat persists unique conversations.
